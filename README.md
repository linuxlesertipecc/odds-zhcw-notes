# odds-zhcw-notes

## 简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存储，可作为轻量级资料查阅或页面展示用途。

仓库不针对任何特定网站或域名，仅作为 HTML 内容的集合与版本管理。

## 目录说明

```
├── pages/          # 存放独立 HTML 页面
├── assets/         # 页面引用的静态资源（CSS、图片、字体等）
├── index.html      # 可选：页面列表或入口页
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名可反映内容主题。
- `assets/`：公共资源文件，页面可按需引用。
- 目录结构可根据实际内容进行扩展或调整。

## 页面归档说明

- 所有页面均为纯前端 HTML，不依赖后端服务。
- 页面内容可包含文字、表格、图表、外部引用等常见元素。
- 归档时建议保持文件命名清晰，避免特殊字符。
- 如有多个页面，可在 `index.html` 中提供导航链接。

## 维护说明

- 本仓库以静态存档为主，不建议频繁修改已发布页面。
- 如需新增页面，请放入 `pages/` 目录并更新入口索引（如有）。
- 资源文件请统一放入 `assets/`，避免分散存放。
- 提交前建议检查 HTML 结构完整性。
- 欢迎通过 Issue 或 Pull Request 提出改进建议。

## 许可

本仓库内容仅供学习与存档参考。如有引用外部资源，请遵守相应版权规定。
